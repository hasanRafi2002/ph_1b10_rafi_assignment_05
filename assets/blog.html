Important FAQ

What is the Document Object Model (DOM)?

The DOM is like a map of a web page that lets you change things like text, images, or buttons using code. It's how browsers understand and interact with the page, making it easy to update or control how the page looks and works.

How do you select an Element from DOM?

To select a DOM element, use:
getElementById("id")
querySelector("selector")
getElementsByClassName("class")
getElementsByTagName("tag")

What is event delegation in the context of DOM, and why is it useful?

Event delegation is when you attach one event listener to a parent element to handle events for its child elements. It's useful because it saves memory and works even for dynamically added elements.

How do you manipulate an element's attributes and styles?

To change an element's attributes and styles in JavaScript, select it by ID and update its attributes or styles as needed.
